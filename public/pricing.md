# Pricing — ShopCommand

Auto repair shop management software. Per shop, not per seat.

## Founding Member (Early Access)
- Price: $100/month for first shop + $50/month for each additional shop
- Trial: 14 days free
- Limits: Unlimited technicians, unlimited users, unlimited repair orders
- Rate lock: Price locked forever as long as subscription is active
- Features: Everything — real-time dashboard, multi-location view, repair order tracking, technician management, customer management, role-based access, mobile tech board, appointments, inspections, file uploads, email notifications, Stripe billing
- Availability: 25 spots total
- Support: Direct access to founder (no ticket queue)

## Standard (After Launch)
- Price: $175/month for first shop + $100/month for each additional shop
- Trial: 14 days free
- Limits: Unlimited technicians, unlimited users, unlimited repair orders
- Features: Same as founding member — all features included
- Support: Standard support

## Notes
- No per-seat fees at any tier
- No feature tiers or paid add-ons — one plan, everything included
- Cancel anytime
- Data export available on cancellation
